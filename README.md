# Todo List Application

Полнофункциональное приложение для управления задачами с фронтендом и бэкендом на Go.

## Технологии

### Backend
- **Go** - основной язык
- **GORM** - ORM для работы с базой данных
- **PostgreSQL** - база данных
- **Docker** - для запуска PostgreSQL

### Frontend
- **HTML5** - структура
- **CSS3** - стилизация с градиентами и анимациями
- **Vanilla JavaScript** - логика приложения

## Структура проекта

```
todo_list/
├── cmd/server/          # Точка входа приложения
├── client/              # Фронтенд (отдельно от бэкенда)
│   ├── index.html       # Главная страница
│   ├── static/
│   │   ├── styles.css   # Стили
│   │   └── app.js       # JavaScript логика
│   └── README.md        # Документация фронтенда
├── internal/
│   ├── api/            # HTTP handlers
│   ├── app/             # Инициализация приложения
│   ├── domain/          # Доменные модели
│   ├── pkg/postgress/   # Подключение к БД и миграции
│   ├── repo/            # Репозитории (GORM)
│   └── usecase/         # Бизнес-логика
├── docker-compose.yaml  # Конфигурация PostgreSQL
└── go.mod               # Зависимости Go
```

## Установка и запуск

### 1. Запуск PostgreSQL

```bash
docker compose up -d
```

PostgreSQL будет доступен на порту `5433` (чтобы не конфликтовать с локальным PostgreSQL).

### 2. Настройка переменных окружения (опционально)

Создайте файл `.env` в корне проекта:

```env
HTTP_PORT=8080
DB_HOST=localhost
DB_PORT=5433
DB_USER=app_user
DB_PASSWORD=app_password
DB_NAME=todo_list
```

Если файл не создан, будут использованы значения по умолчанию.

### 3. Запуск приложения

```bash
go run cmd/server/main.go
```

Приложение будет доступно по адресу: http://localhost:8080

## API Endpoints

- `POST /api/todo` - Создать новую задачу
  ```json
  {
    "body": "Текст задачи"
  }
  ```

- `GET /api/todo` - Получить все задачи

- `PATCH /api/todo/{id}` - Отметить задачу как выполненную

- `DELETE /api/todo/{id}` - Удалить задачу

## Функционал фронтенда

- ✅ Добавление новых задач
- ✅ Просмотр списка задач
- ✅ Отметка задач как выполненных
- ✅ Удаление задач
