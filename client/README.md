# Frontend - Todo List

Фронтенд приложения Todo List.

## Структура

```
client/
├── index.html          # Главная страница
└── static/
    ├── styles.css     # Стили
    └── app.js         # JavaScript логика
```

## Технологии

- **HTML5** - структура
- **CSS3** - стилизация с градиентами и анимациями
- **Vanilla JavaScript** - логика приложения (без фреймворков)

## Функционал

- ✅ Добавление новых задач
- ✅ Просмотр списка задач
- ✅ Отметка задач как выполненных
- ✅ Удаление задач
- ✅ Статистика (всего, выполнено, осталось)
- ✅ Адаптивный дизайн
- ✅ Красивый современный интерфейс

## API

Фронтенд работает с бэкенд API через следующие endpoints:

- `POST /api/todo` - Создать задачу
- `GET /api/todo` - Получить все задачи
- `PATCH /api/todo/{id}` - Отметить как выполненную
- `DELETE /api/todo/{id}` - Удалить задачу

## Разработка

Фронтенд раздается бэкендом автоматически. Для разработки:

1. Запустите бэкенд: `go run cmd/server/main.go`
2. Откройте в браузере: http://localhost:8080

Файлы автоматически подхватываются из папки `client/`.
